# YEET

Version 0.0.1  
For Game Version 0.26.2  
Developed by 18107  
Requires VoidManager 1.1.4


---------------------

### 💡 Function - Throw all items with maximum force without charging the throw

### 🎮 Client Usage

- Install
- Configure at F5 > Mod Settings > YEET
- Violently throw objects

### 👥 Multiplayer Functionality

- ✅ Client
  - This does not affect other players

---------------------

## 🔧 Install Instructions - **Install following the normal BepInEx procedure.**

Ensure that you have [BepInEx 5](https://thunderstore.io/c/void-crew/p/BepInEx/BepInExPack/) (stable version 5 **MONO**) and [VoidManager](https://thunderstore.io/c/void-crew/p/VoidCrewModdingTeam/VoidManager/) installed.

#### ✔️ Mod installation - **Unzip the contents into the BepInEx plugin directory**

Drag and drop `YEET.dll` into `Void Crew\BepInEx\plugins`
